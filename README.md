# Quant Support Analyst Portfolio — Advanced Projects

This repository extends the starter portfolio with 5 additional public, recruiter-friendly projects for roles such as:

- Quantitative Support Analyst
- Investment Analyst
- Portfolio Analyst
- Trading Data Analyst
- Financial Data Scientist

The focus is practical: market questions, clean Python notebooks, readable reasoning, and outputs that could help a trader, portfolio manager, or investment team.

> Data source: Yahoo Finance via `yfinance` where applicable. Some option projects use manually entered example prices because free option-chain data is often limited and inconsistent. This repository is for educational and portfolio purposes only. It is not investment advice.

## Projects included

| Notebook | Project | What it demonstrates |
|---|---|---|
| `04_options_strategy_analyzer.ipynb` | Options payoff analyzer | Options logic, risk/reward, payoff diagrams, trader support tooling |
| `05_portfolio_risk_dashboard.ipynb` | Portfolio risk dashboard | Portfolio analytics, volatility, drawdown, correlation, allocation |
| `06_market_breadth_dashboard.ipynb` | Market breadth dashboard | Market internals, sector leadership, trend participation |
| `07_earnings_reaction_study.ipynb` | Earnings reaction study | Event-study thinking, volatility context, hypothesis testing |
| `08_volatility_regime_backtest.ipynb` | Volatility regime backtest | Regime analysis, VIX filters, systematic testing |
